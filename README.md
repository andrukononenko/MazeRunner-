🧩 MazeRunner – Лабиринт на 7 языках
Классическая игра «Лабиринт» в терминале с процедурной генерацией, поиском пути, таймером и рекордами.
Найди выход из лабиринта, собирай монеты, улучшай свои навыки и соревнуйся с собой!
Реализована на 7 языках программирования с единым интерфейсом и богатым функционалом.

🚀 Возможности
Генерация лабиринта – алгоритм Depth‑First Search с рекурсивным бэктрекингом создаёт уникальные лабиринты каждый раз.

Управление – стрелки (←↑↓→) или клавиши WASD для перемещения игрока.

Цветной вывод – стены, пути, игрок и выход выделены разными цветами.

Счётчик шагов – показывает количество сделанных ходов.

Таймер – засекает время прохождения уровня.

Рекорды – лучшее время и минимальное число шагов сохраняются в локальный файл.

Несколько уровней – размер лабиринта увеличивается с каждым пройденным уровнем (опционально).

Сбор монет – на пути разбросаны монеты (💎), которые нужно собрать для получения бонусных очков.

Подсветка пути – при желании можно показать кратчайший путь (опция --show-path).

Интерактивность – игра реагирует на нажатия клавиш без Enter (raw mode).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --size <N>	Размер лабиринта (N×N), по умолчанию 10 (чётное число)
-c, --coins	Количество монет (по умолчанию 5)
-p, --show-path	Показывать кратчайший путь к выходу (подсветка)
-h, --help	Справка
Примеры
bash
# Запустить игру с лабиринтом 10×10
python maze.py

# Лабиринт 20×20 с 10 монетами
python maze.py -s 20 -c 10

# Показать кратчайший путь
python maze.py -p
Управление
↑ / W – движение вверх

↓ / S – движение вниз

← / A – движение влево

→ / D – движение вправо

Q – выход из игры

R – перезапуск уровня (в некоторых версиях)

🧠 Алгоритм генерации лабиринта
Лабиринт представляет собой сетку (N×N) с чётными размерами.

Стены находятся на нечётных координатах, проходы – на чётных.

Используется рекурсивный бэктрекинг:

Начинаем с клетки (1,1).

Выбираем случайного соседа через две клетки (прыжок через стену).

Убираем стену между текущей и выбранной клеткой.

Рекурсивно переходим в выбранную клетку.

Выход всегда находится в правом нижнем углу (N-2, N-2).

Игрок начинает в левом верхнем углу (1,1).

✨ Дополнительные фичи
Сбор монет – на пути разбросаны монеты (символ 💎), которые добавляют 10 очков к финальному счёту.

Подсветка пути – при включённой опции --show-path кратчайший путь к выходу отображается зелёными точками.

Рекорды – сохраняются в файл ~/.maze_records.json и показываются при завершении игры.

Автоматическое масштабирование – лабиринт подстраивается под размер терминала (если он слишком мал).

Анимация – при движении игрок оставляет след (в некоторых реализациях).
